Methods:

JWT Token Endpoint

Info

Alternatively you can generate JWT Token Pairs with your user credentials to use instead of an API Key for authentication.


Token Generate Endpoint: https://evntmngr.xyz/api/auth/
Token Refresh Endpoint: https://evntmngr.xyz/api/auth/refresh/

Methods

POST: Generate a JWT Token Pair or Refresh JWT Access Token
Authentication: Pass your user credentials in the headers

The Access Token is valid for 30 mins and the Refresh Token is valid for 24 hours.

POST - Auth

POST https://evntmngr.xyz/api/auth/
Content-Type: application/json

{
"email": "your-email",
"password": "your-password"
}

POST - Refresh

POST https://evntmngr.xyz/api/auth/refresh/
Content-Type: application/json

{
"refresh": "your-refresh-token"
}
